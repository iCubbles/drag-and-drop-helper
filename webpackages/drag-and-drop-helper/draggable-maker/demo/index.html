dragable-maker

Not visible Element, no demo available.

Instructions

  • use it in compound component
  • make a sibling member content draggable
  • initialise the slot "draggableElementSelector" with a css selector of the element want to make it draggable. The selector will used in the context of the (parent) compound component.
  • by drag the draggable element will the runtimeId of the parent compound component transported as data.
